techminer.analytics.count_terms(x, column)[source]¶

Counts the number of different terms in a column.

Parameters

  • x (pandas.DataFrame) – Biblographic dataframe.

  • column (str) – the column to explode.

  • sep (str) – Character used as internal separator for the elements in the column.

Returns

DataFrame.

Examples

>>> x = pd.DataFrame({'Authors': ['xxx', 'xxx; zzz', 'yyy', 'xxx; yyy; zzz']})
>>> x
         Authors
0            xxx
1       xxx; zzz
2            yyy
3  xxx; yyy; zzz

>>> count_terms(x, column='Authors')
3

techminer.analytics.summary_by_term(x, column, keywords=None)[source]¶

Summarize the number of documents and citations by term in a dataframe.

Parameters

  • column (str) – the column to explode.

  • keywords (int, list) – filter the results.

Returns

DataFrame.

Examples

>>> import pandas as pd
>>> x = pd.DataFrame(
...     {
...          "Authors": "author 0;author 1;author 2,author 0,author 1,author 3".split(","),
...          "Cited by": list(range(10,14)),
...          "ID": list(range(4)),
...     }
... )
>>> x
                      Authors  Cited by  ID
0  author 0;author 1;author 2        10   0
1                    author 0        11   1
2                    author 1        12   2
3                    author 3        13   3

>>> summary_by_term(x, 'Authors')
    Authors  Num Documents  Cited by      ID
0  author 0              2        21  [0, 1]
1  author 1              2        22  [0, 2]
2  author 2              1        10     [0]
3  author 3              1        13     [3]

>>> keywords = Keywords(['author 1', 'author 2'])
>>> keywords = keywords.compile()
>>> summary_by_term(x, 'Authors', keywords=keywords)
    Authors  Num Documents  Cited by      ID
0  author 1              2        22  [0, 2]
1  author 2              1        10     [0]
techminer.analytics.summary_by_year(df)[source]¶

Computes the number of document and the number of total citations per year. This funciton adds the missing years in the sequence.

Parameters

df (pandas.DataFrame) – bibliographic dataframe.

Returns

pandas.DataFrame.

Examples

>>> import pandas as pd
>>> df = pd.DataFrame(
...     {
...          "Year": [2010, 2010, 2011, 2011, 2012, 2016],
...          "Cited by": list(range(10,16)),
...          "ID": list(range(6)),
...     }
... )
>>> df

0  2010        10   0
1  2010        11   1
2  2011        12   2
3  2011        13   3
4  2012        14   4
5  2016        15   5
