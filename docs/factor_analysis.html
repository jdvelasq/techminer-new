Factor analysis¶

techminer.factor_analysis.app(data, limit_to=None, exclude=None, tab=None)[source]¶

Jupyter Lab dashboard.

techminer.factor_analysis.factor_analysis(data, column, n_components=2, output=0, top_by=None, top_n=None, sort_by=None, ascending=True, figsize=10, 10, layout='Kamada Kawai', cmap=None, limit_to=None, exclude=None)[source]¶

Computes the matrix of factors for terms in a given column.

Parameters

  • data (pandas.DataFrame) – A bibliographic dataframe.

  • column (str) – column to process.

  • n_components (int) – Number of components to compute.

  • output (int/str) –

    Output generated by the function:

    • 0 or ‘Matrix’.

    • 1 or ‘Network’.

  • top_by (int/str) –

    Criteria for selecting the elements of the matrix.

    • -1 or ‘Values’: values >= 0.25 or values <= -0.25.

    • 0 or ‘Num_Documents’: top terms based on frequency.

    • 1 or ‘Times_Cited’: top terms based on total citations.

  • sort_by (str) –

    • ‘Alphabetic’.

    • ’Num Documents’.

    • ’Times Cited’.

    • ’F#’.

  • ascending (bool) – Sort ascending vs. descending.

  • figsize (tuple) – Figure dimensions.

  • layout (str) –

    • “Circular”.

    • ”Kamada Kawai”

    • ”Planar”.

    • ”Spectral”.

    • ”Spring”.

    • ”Shell”.

  • cmap (str) – Matplotlib’s colormap.

  • limit_to (dict) – Dictionary to ‘limit to’ terms in the output.

  • exclude (dict) – Dictionary to specify terms to ignore.

Returns

DataFrame.

Examples

>>> import pandas as pd
>>> x = [ 'A', 'A;B', 'B', 'A;B;C', 'B;D', 'A;B']
>>> y = [ 'a', 'a;b', 'b', 'c', 'c;d', 'd']
>>> df = pd.DataFrame(
...    {
...       'Authors': x,
...       'Author_Keywords': y,
...       "Times_Cited": list(range(len(x))),
...       'ID': list(range(len(x))),
...       'Year': [1990, 1990, 1991, 1991, 1992, 1992],
...    }
... )
>>> df
