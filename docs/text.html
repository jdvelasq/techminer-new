Text manipulation¶

This module contains functions for manipulating texts.

Functions in this module¶

techminer.text.find_string(patterns, x, ignore_case=True, full_match=False, use_re=False, explode=True)[source]¶

Find patterns in the elements of a list.

>>> x = ['aa;b', 'c;d', 'A', 'e', None]
>>> find_string('a', x)
['A', 'aa']

>>> find_string('a', x, ignore_case=False)
['aa']

>>> find_string('a', x, full_match=True)
['A']
techminer.text.fingerprint(x)[source]¶

Computes ‘fingerprint’ representation of string x.

See https://github.com/OpenRefine/OpenRefine/wiki/Clustering-In-Depth

Parameters

x (string) – string to convert.

Returns

string.

Examples

>>> fingerprint('a A b')
'a b'
>>> fingerprint('b a a')
'a b'
>>> fingerprint(None) is None
True
>>> fingerprint('b c')
'b c'
>>> fingerprint(' c b ')
'b c'
